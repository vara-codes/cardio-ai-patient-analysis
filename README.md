# 🫀 Cardio AI - Patient Data Analysis

## 📌 Overview

This project demonstrates Exploratory Data Analysis (EDA) on a cardiology patient dataset using **Python** and **Pandas**.

The project covers data cleaning, patient analysis, aggregation, filtering, and statistical insights.

---

## 🚀 Technologies Used

- Python
- Pandas
- Jupyter Notebook

---

## 📂 Dataset Features

The dataset contains information such as:

- Patient ID
- Name
- Age
- Gender
- Heart Rate
- Blood Pressure
- Cholesterol
- BMI
- Oxygen Level
- ECG Status
- Risk Category
- Admission Type
- Hospital Stay Days

---

## 📊 Analysis Performed

- Dataset overview
- Missing value handling
- Duplicate record detection
- Data filtering
- Sorting
- Unique value analysis
- Risk category analysis
- HealthScore calculation
- GroupBy operations
- Aggregation using `agg()`
- Statistical insights
